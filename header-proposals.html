The Agentic Web

31 Jan 2026

We're entering a new era of the web — one where AI agents don't just assist humans, they navigate, negotiate, and transact on our behalf. This shift is as fundamental as the move from desktop to mobile.

Think about what happened with mobile. We didn't just shrink websites — we rebuilt the entire interaction model. Touch replaced clicks. Apps replaced bookmarks. Location became a first-class input. The "mobile web" wasn't the web made smaller; it was a new medium.

The agentic web will be similarly transformative. When your AI agent visits a website, it doesn't need beautiful typography or clever animations. It needs structured data, clear capabilities, and machine-readable intent.

What changes

First, discovery changes completely. Today, SEO is about ranking in a list of ten blue links. Tomorrow, it's about being the service an agent selects from a capability registry. The competitive moat shifts from brand recall to API quality.

The best interface for an AI agent is no interface at all — just a well-documented protocol.

Second, trust and authentication become critical infrastructure. When an agent books a flight on your behalf, how does the airline know it's authorized? How do you verify the agent chose the best option? We need new primitives for delegated authority.

Third, the economics of attention collapse. If agents handle routine browsing, the remaining human-directed web traffic becomes extremely high-intent. Every click becomes more valuable because it represents a deliberate choice to engage personally rather than delegate.

Building for agents

At Canva, we've been thinking about this from both sides. Our Magic Studio tools help humans create — but increasingly, agents need to create too. An agent scheduling social media posts needs to generate graphics. An agent planning an event needs to design invitations.

The pattern we're seeing is that agents want the same capabilities as humans, but through different interfaces. They don't need drag-and-drop; they need parameters. They don't need previews; they need guarantees about output quality.

This is why I created the agents.json specification — a simple way for websites to declare their capabilities to AI agents. Think of it as robots.txt for the agentic era, but instead of saying "don't crawl this," it says "here's what I can do for you."

The web has always been about connecting humans to information and to each other. The agentic web extends this to connecting humans to capabilities, mediated by intelligent agents that understand our preferences and constraints. It's not the end of the human web — it's the beginning of something much larger.
